# Content Notes: K.CC.B.5 (How Many? 0-10)

ใช้ไฟล์นี้เป็น “สคริปต์เนื้อหา” ก่อนทำเป็นงานกราฟิกจริง

## Product

- Standard: `K.CC.B.5`
- Skill: How many? (count to answer)
- Range: 0-10
- Theme: none (generic)

## Pages

1. Cover: K.CC.B.5 How Many? (0-10)
2. Instructions: Count the objects. Write the number.
3. Practice 1: objects in a row (1-5) - count & write
4. Practice 2: objects in a row (6-10) - count & write
5. Practice 3: scattered objects (1-5) - count & write
6. Practice 4: scattered objects (6-10) - count & write
7. Practice 5: two groups (1-5 each) - write each total (no adding)
8. Practice 6: count & circle the number (multiple choice, 3 options)
9. Review: mixed (row + scattered + one two-groups item)
10. Answer key
11. Credits

## Answer Key Rule

- เฉลยคือ “จำนวนวัตถุทั้งหมด” ที่อยู่ในรูปของข้อนั้น
- ตรวจ 2 รอบ: (1) จำนวนในรูปตรงกับเฉลย (2) ทุกข้ออยู่ในช่วง 0-10
