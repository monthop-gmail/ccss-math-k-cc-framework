## 🟢 **K.CC.A.1 — Count to 100 by ones and tens**

**ความหมาย:**
 เด็กต้องสามารถนับเลข 1–100

- 

นับทีละ 1 (1,2,3…)

- 

นับทีละ 10 (10,20,30…)

**ทักษะที่ต้องการ**

- 

ลำดับตัวเลข

- 

ความจำเชิงลำดับ

- 

เข้าใจ pattern ของเลข

**ตัวอย่างโจทย์**

- 

Fill in the missing numbers: 21, 22, __, 24

- 

Count by tens: 10, __, 30, __, 50

- 

Circle the number that comes next: 47 → ?

---

## 🟢 **K.CC.A.2 — Count forward beginning from a given number**

**ความหมาย:**
 เด็กต้องเริ่มนับจากเลขที่กำหนด (ไม่ใช่เริ่มจาก 1 เสมอ)

**ตัวอย่างโจทย์**

- 

Start at 6. Count: 6, __, __, __

- 

Continue counting: 13, 14, __, __

- 

Teacher says “Start at 9” → student counts aloud

---

## 🟢 **K.CC.A.3 — Write numbers from 0 to 20**

**ความหมาย:**
 เด็กต้องเขียนเลข 0–20 ได้ถูกต้อง

**ตัวอย่างโจทย์**

- 

Trace and write numbers 11–15

- 

Write the number that matches the picture (12 apples → write 12)

- 

Dictation: teacher says number → student writes

## 🟡 **K.CC.B.4 — Understand the relationship between numbers and quantities**

มาตรฐานนี้มี 3 ส่วนย่อยสำคัญ

### 🔹 (a) One-to-one correspondence

นับวัตถุแล้วชี้ทีละชิ้น

**โจทย์**

- 

Count and touch each star

- 

Count and color the correct number of circles

---

### 🔹 (b) Last number = quantity

เลขสุดท้ายที่นับ = จำนวนทั้งหมด

**โจทย์**

- 

Count the bears. How many?

- 

Student counts blocks and says final number

---

### 🔹 (c) Each successive number is one more

เข้าใจว่าเลขถัดไปมากกว่า 1

**โจทย์**

- 

Which is more: 5 or 6?

- 

Show one more cube than 7

---

## 🟡 **K.CC.B.5 — Count to answer “How many?”**

**ความหมาย:**
 เด็กต้องนับวัตถุเพื่อหาจำนวน (สูงสุด 20)

**ตัวอย่างโจทย์**

- 

How many balloons?

- 

Count the animals and write the number

- 

Count scattered objects

**ทักษะสำคัญ**

- 

นับของเรียง

- 

นับของกระจาย

- 

นับของหลายกลุ่ม

## 🔵 **K.CC.C.6 — Identify whether one group has more, less, or equal**

**ความหมาย:**
 เปรียบเทียบจำนวน 2 กลุ่ม

**ตัวอย่างโจทย์**

- 

Circle the group with more apples

- 

Which has less?

- 

Same or different amount?

**รูปแบบใบงานยอดนิยม**

- 

Count then compare

- 

Draw to match

- 

Cut and paste comparison

## 🔵 **K.CC.C.7 — Compare two numbers between 1 and 10**

**ความหมาย:**

**ตัวอย่างโจทย์**
