# Thumbnail / Preview Notes (TPT)

## Minimum Set (แนะนำ)

- Cover (ใหญ่ ชัด)
- "What's Included" (จำนวนหน้า + มี answer key)
- 2-4 หน้า sample (โชว์รูปแบบโจทย์หลากหลาย)
- Answer key sample (1 หน้า)

## Copy ที่ควรมีบนภาพ

- `CCSS K.CC.__.__` (ให้คนเห็นรหัสทันที)
- Range: `0-10`, `0-20`, `1-10`
- Pages count เช่น `10 Pages + Answer Key`

## Layout Tips

- ใช้ตัวอักษรใหญ่ อ่านได้บนมือถือ
- เน้น 1-2 สีหลักให้เป็นแบรนด์ และคุมความรก
