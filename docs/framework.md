# Framework (K.CC Product System)

เอกสารนี้เป็นภาพรวม framework ในการแตกมาตรฐาน K.CC ให้กลายเป็น “product line” ที่ผลิตซ้ำได้

## Inputs

- Standard: `K.CC.A.1` / `K.CC.A.2` / `K.CC.A.3` / `K.CC.B.4` / `K.CC.B.5` / `K.CC.C.6` / `K.CC.C.7`
- Range: 0–10 / 0–20 / 1–10 / 1–100 (ตามมาตรฐาน)
- Skill focus: เช่น one-to-one, cardinality, compare, count forward
- Theme: animals/food/space/seasonal
- Format: worksheet / centers / assessment / intervention / challenge

## Outputs (ต่อ 1 product)

- PDF ใบงาน (practice 5–10 หน้า)
- Answer key
- Cover + credits
- Listing copy (Title/Description/Keywords)
- Thumbnail set

## Reuse Strategy

- ทำ `templates/` ให้ครบ 1 ชุด แล้ว copy ไปสร้างสินค้าใหม่โดยเปลี่ยนแค่: numbers + art/theme + prompt
