# TPT SEO Notes (K.CC)

เป้าหมาย: ให้คนค้นเจอจาก "standard code" + "skill keyword" + "range" แล้วคลิกซื้อ

## 1) Naming Formula

- Title: `CCSS [Standard] [Skill] [Range] Worksheets (Theme)`
- ตัวอย่าง: `CCSS K.CC.C.7 Compare Numbers 1-10 Worksheets (Animals)`

## 2) Keyword Buckets (ใส่ให้ครบ)

- Standard code: `K.CC.B.5`, `K.CC.C.7` (ต้องอยู่ทั้ง Title/Description/Keywords)
- Grade: `Kindergarten math`
- Skill: `counting`, `how many`, `cardinality`, `compare numbers`, `greater than`, `less than`
- Use case: `math centers`, `morning work`, `small groups`, `intervention`, `independent practice`
- Range: `0-10`, `0-20`, `1-10`
- Theme (ถ้ามี): `animals`, `space`, `farm`, `ocean`

## 3) Description (สั้น แต่ครบ)

- 2 บรรทัดแรกต้องบอก: มาตรฐาน + เด็กทำอะไรได้
- ใส่ bullet สั้น ๆ: Students will practice / Includes / Perfect for
- ใส่ "File type" และขนาดกระดาษ (US Letter) ชัดเจน

## 4) Bundle Strategy (เร็วและปลอดภัย)

- ทำ bundle ทุก 6-8 products (ชื่อ bundle ก็ใส่ standard codes ที่ครอบคลุม)
- ใช้ preview/thumbnail template เดิมเพื่อให้ร้านดูเป็นชุดเดียวกัน
