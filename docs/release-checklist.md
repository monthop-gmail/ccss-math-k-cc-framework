# Release Checklist (Pack -> TPT)

ใช้เช็ครอบสุดท้ายก่อนอัปโหลด (นอกเหนือจาก `templates/quality-checklist.md`)

## Files

- PDF เปิดได้ ไม่มีฟอนต์เพี้ยน
- มี Credits page + ข้อมูล licensing ครบ
- ตั้งชื่อไฟล์ PDF ตาม convention (มี standard code)

## Listing

- Title มี `CCSS` + standard code + skill + range
- Description บรรทัดแรกมี standard code ชัด
- Keywords มี: standard code + grade + skill + range + use case

## Thumbnails / Preview

- ภาพ cover อ่านชัดบนมือถือ
- มีภาพ "What's included" และตัวอย่างหน้าใบงาน
- ใส่ range และ "Answer Key" บนภาพอย่างน้อย 1 จุด

## QA

- สะกดคำ/ตัวเลข/สัญลักษณ์ถูกต้อง
- ไม่มี prompt ที่กำกวม (เด็กทำได้ด้วยตัวเอง)
