# Blueprint: K.CC.C.7 (Compare Numbers 1-10)

Target: Compare two numbers between 1 and 10 using `> < =`

## Core Variants (แตกเป็นสินค้า)

- Numbers 1-5 (easy)
- Numbers 1-10 (core)
- With visuals (compare quantities) vs. Numbers only (symbol practice)
- Cut & paste symbols vs. Write the symbol

## Page Set (แนะนำ)

- Cover
- Instructions + mini anchor chart `> < =` (1 page)
- Practice (8 pages)
- Review (1 page)
- Answer key (1-2 pages)
- Credits

## Practice Patterns (เลือก 2-4 แบบวน)

- Write `> < =` between two numbers
- Circle the bigger/smaller number
- Match pairs that are equal
- (Optional) Visual support: ten frames/dots then compare

## Common Mistakes (QC)

- สัญลักษณ์ `>`/`<` กลับด้านในตัวอย่าง
- ใช้เลข 0 หรือ 11+ (หลุดช่วง 1-10)
