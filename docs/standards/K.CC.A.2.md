# Blueprint: K.CC.A.2 (Count forward from a given number)

Target: Start counting from a given number (not always from 1)

## Core Variants (แตกเป็นสินค้า)

- Start at (0-10) then count on
- Start at (0-20) then count on
- Mixed "continue counting" strips

## Practice Patterns

- `Start at 6: 6, __, __, __`
- `Continue: 13, 14, __, __`
- Number line jumps (small steps)

## Design Tips

- ให้เริ่มจากเลขที่เห็นชัด (วงกลม/ไฮไลต์จุดเริ่ม)
- จำกัดจำนวนช่องเติมต่อหนึ่งข้อ (3-6 ช่อง)

## Common Mistakes (QC)

- โจทย์ให้เริ่มนับจากเลขที่ไม่เหมาะกับช่วงที่ประกาศไว้
- จำนวนช่องเติมยาวเกิน ทำให้กลายเป็น K.CC.A.1 แบบยาว ๆ
