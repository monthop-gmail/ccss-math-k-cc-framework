# Blueprint: K.CC.B.4 (Numbers and quantities)

Target: One-to-one correspondence + cardinality + understanding "one more"

## Sub-skills

- (a) Count objects with one-to-one correspondence
- (b) Cardinality: last number said = how many
- (c) Successive numbers: each next number = one more

## Core Variants (แตกเป็นสินค้า)

- Count & touch (mark each object once)
- Cardinality check: "How many?" after counting
- One more / one less (within 0-10)

## Practice Patterns

- Count and mark (dot/circle) each object once, then write total
- After counting: "How many?" (same picture, different prompt)
- "Show one more" (draw one more object)

## Common Mistakes (QC)

- วัตถุชิดกันจนเด็กแยกชิ้นไม่ได้
- สั่งงานหลายขั้นเกินไปใน 1 prompt
