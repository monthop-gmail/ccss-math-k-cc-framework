# Blueprint: K.CC.A.1 (Count to 100 by ones and tens)

Target: Count to 100 by ones and by tens

## Core Variants (แตกเป็นสินค้า)

- Count by ones (1-100) missing numbers
- Count by tens (10-100) fill in the pattern
- Mixed practice (ones + tens) review pack

## Page Set (แนะนำ)

- Cover
- Instructions (1 page)
- Practice (8-10 pages)
- Review (1 page)
- Answer key (1-2 pages)
- Credits

## Practice Patterns

- Fill in missing numbers (within 1-30, 1-50, 1-100)
- Count by tens: `10, __, 30, __, 50`
- What comes next / before (limited range)

## Differentiation

- Easy: 1-30, fewer blanks, number line support
- Core: 1-100, mixed blanks
- Challenge: mixed sequences, two-step patterns

## Common Mistakes (QC)

- ช่องว่างเยอะเกินสำหรับ K (ทำให้เดายาก/ท้อ)
- มีเลขข้ามลำดับ/ตัวสะกดผิดในโจทย์
