# Blueprint: K.CC.A.3 (Write numbers 0-20)

Target: Write numbers from 0 to 20

## Core Variants (แตกเป็นสินค้า)

- Tracing numbers 0-10
- Tracing numbers 0-20
- Write the number to match quantity (0-10 / 0-20)

## Practice Patterns

- Trace then write (2-3 times per number)
- Write the number shown (no tracing)
- Count objects then write the number

## Design Tips

- ใช้ฟอนต์ตัวเลขที่อ่านง่าย และพื้นที่เขียนพอ
- สำหรับ 0 และ 1-2 ให้มีตัวอย่างชัดเพื่อกันกลับด้าน

## Common Mistakes (QC)

- เส้น trace แน่นเกิน/เล็กเกิน
- จำนวนวัตถุไม่ตรงกับคำตอบในหน้า match quantity
