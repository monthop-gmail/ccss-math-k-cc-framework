# Blueprint: K.CC.B.5 (How Many?)

Target: Count to answer "How many?" up to 20 (objects in row, scattered, multiple groups)

## Core Variants (แตกเป็นสินค้า)

- 0-10 (Quick Win)
- 0-20
- Row only / Scattered only
- Mixed groups (2 groups) / Mixed page

## Page Set (แนะนำ)

- Cover
- Instructions (1 page)
- Practice (8 pages)
- Review (1 page)
- Answer key (1-2 pages)
- Credits

## Practice Patterns (เลือก 3-4 แบบวน)

- Count & write the number
- Count (scattered) & write
- Count two groups (write each total)
- Count & circle (multiple choice)

## Common Mistakes (QC)

- จำนวนวัตถุในรูปไม่ตรงกับเฉลย
- ขอบเขตเลขหลุด (เช่นมี 11 ใน pack 0-10)
- รูปเล็ก/ชิดกันเกิน ทำให้นับยากสำหรับ K
