# Blueprint: K.CC.C.6 (More / Less / Equal)

Target: Compare two groups to decide more, less, or equal

## Core Variants (แตกเป็นสินค้า)

- Compare quantities (1-10) with pictures
- Draw to make equal
- Cut & paste: more/less/equal sort

## Practice Patterns

- Count both groups, then circle: more / less / equal
- Draw objects to match a target group (make equal)
- "Which has more?" with simple visuals (ten frames/dots)

## Common Mistakes (QC)

- ใช้ภาพที่หลอกตา (วัตถุใหญ่/เล็ก) จนกลายเป็น compare size ไม่ใช่จำนวน
- สุ่มจำนวนที่ต่างกันมากเกิน (เด็กนับไม่ทัน)
