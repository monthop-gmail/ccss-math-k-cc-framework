# Standards Index (K.CC)

ใช้ไฟล์ในโฟลเดอร์นี้เป็น "blueprint" ก่อนเริ่มทำ pack จริง

- `K.CC.A.1.md` Count to 100 by ones and tens
- `K.CC.A.2.md` Count forward from a given number
- `K.CC.A.3.md` Write numbers 0-20
- `K.CC.B.4.md` Number-quantity relationship (one-to-one + cardinality)
- `K.CC.B.5.md` How many? (count up to 20)
- `K.CC.C.6.md` More / Less / Equal (compare groups)
- `K.CC.C.7.md` Compare numbers 1-10 (`> < =`)
