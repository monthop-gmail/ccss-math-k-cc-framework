Project Structure

- `docs/` แนวคิด framework + workflow
- `templates/` ไฟล์แม่แบบสำหรับทำสินค้า TPT
- `examples/` ตัวอย่าง K.CC แบบแพ็กขาย

Quick Start

1) เลือก standard จาก `docs/standards/`
2) Copy โครงจาก `templates/product-pack/` ไปสร้างแพ็กใหม่
3) เขียน listing ด้วย `templates/tpt-listing-template.md` หรือ `templates/product-pack/listing/listing.md`
