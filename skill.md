# 🧠 CCSS Math K.CC Framework Skill

A ready-to-use AI skill for generating high-converting TPT products aligned with Common Core Kindergarten Counting & Cardinality (K.CC) standards.

## ✅ Capabilities
- Auto-generate TPT-ready worksheets per standard (`K.CC.A.1`, `K.CC.B.4`, etc.)
- Create SEO-optimized listing titles & descriptions using official standard codes and high-volume keywords
- Produce theme-based variants (Animals, Space, Ocean, etc.) from a single template
- Build bundles (6–8 items), review packs, and assessment kits
- Enforce production rules: ≤3 hrs/product, no >2-day upload gaps, bundle every 6–8 items

## 📁 Key Assets
- `templates/product-pack/` — Reusable layout (instruction, 5–10 practice pages, review, answer key)
- `docs/standards/` — Blueprint per standard with example problems & cognitive focus
- `examples/` — Real-world TPT-style samples (e.g., `K.CC.B.5_HowMany_0-20.zip`)

## 🚀 How to Use
1. Specify target standard (e.g., `K.CC.C.7`) or phase (e.g., "Phase 2: Theme Expansion")
2. Choose theme (optional) and output format (PDF draft / Markdown listing / ZIP package)
3. AI generates compliant, classroom-ready assets instantly

> 💡 Pro Tip: Start with Phase 1 Quick Wins (`K.CC.B.5`, `K.CC.C.7`, `K.CC.A.3`) for fastest traction.
