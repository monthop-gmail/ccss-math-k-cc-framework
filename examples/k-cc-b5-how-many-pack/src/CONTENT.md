# Content Notes (Editable)

ใช้ไฟล์นี้เป็น “สคริปต์เนื้อหา” ก่อนทำเป็นงานกราฟิกจริง

## Pages

1. Cover: K.CC.B.5 How Many? (0–10)
2. Instructions: Count the objects. Write the number.
3. Practice 1: objects in a row (1–5)
4. Practice 2: objects in a row (6–10)
5. Practice 3: scattered objects (1–5)
6. Practice 4: scattered objects (6–10)
7. Practice 5: two groups (add nothing, just count each group)
8. Practice 6: count & circle the number (multiple choice)
9. Review: mixed page
10. Answer key

## Answer Key Rule

- ใส่เฉลยเป็น “ตัวเลขสุดท้ายที่นับได้” และตรวจว่า count ตรงกับรูปทุกครั้ง
