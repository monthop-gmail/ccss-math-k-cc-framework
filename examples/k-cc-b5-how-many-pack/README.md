# Example Pack: K.CC.B.5 (How Many?)

ตัวอย่างโครงแพ็กสำหรับขายบน TPT (ยังไม่ใช่ไฟล์กราฟิกจริง)

## Target Standard

- `K.CC.B.5` Count to answer “How many?” (0–20)

## Suggested Product Variants

- 0–10 (Quick Win)
- 0–20
- Scattered objects only
- Mixed groups

## Folder Layout (แนะนำ)

- `src/` ไฟล์ทำงาน (แก้ไขได้)
- `export/` ไฟล์ส่งออก (PDF)
- `listing/` title/description/keywords
- `thumbnails/` ภาพหน้าปก/พรีวิว

## Practice Page Ideas (เชิงโจทย์)

- Count the balloons. Write the number.
- Count the animals (scattered). Circle the correct number.
- Count two groups. Write totals.
- Challenge: Count and show one more.
