# Example Pack: K.CC.B.4 (One-to-One + Cardinality)

ตัวอย่างแพ็กเน้น one-to-one correspondence และ cardinality (ยังไม่ใช่ไฟล์กราฟิกจริง)

## Target Standard

- `K.CC.B.4` Understand the relationship between numbers and quantities

## Suggested Variants

- Count & mark (0-10)
- Cardinality "How many?" (0-10 / 0-20)
- One more (within 0-10)

## Folder Layout

- `src/` ไฟล์ทำงาน (แก้ไขได้)
- `export/` ไฟล์ส่งออก (PDF)
- `listing/` title/description/keywords
- `thumbnails/` ภาพหน้าปก/พรีวิว
