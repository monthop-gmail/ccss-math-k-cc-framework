# Example Pack: K.CC.C.6 (More / Less / Equal)

ตัวอย่างแพ็กเปรียบเทียบ "กลุ่ม" ว่ามากกว่า/น้อยกว่า/เท่ากัน (ยังไม่ใช่ไฟล์กราฟิกจริง)

## Target Standard

- `K.CC.C.6` Identify whether the number of objects in one group is greater than, less than, or equal

## Suggested Variants

- Compare groups 1-5 (easy)
- Compare groups 1-10 (core)
- Draw to make equal

## Folder Layout

- `src/` ไฟล์ทำงาน (แก้ไขได้)
- `export/` ไฟล์ส่งออก (PDF)
- `listing/` title/description/keywords
- `thumbnails/` ภาพหน้าปก/พรีวิว
