# Example Pack: K.CC.A.3 (Write Numbers 0-20)

ตัวอย่างแพ็กสำหรับฝึกเขียนเลข 0-20 (ยังไม่ใช่ไฟล์กราฟิกจริง)

## Target Standard

- `K.CC.A.3` Write numbers from 0 to 20

## Folder Layout

- `src/` ไฟล์ทำงาน (แก้ไขได้)
- `export/` ไฟล์ส่งออก (PDF)
- `listing/` title/description/keywords
- `thumbnails/` ภาพหน้าปก/พรีวิว
