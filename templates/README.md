# Templates

โฟลเดอร์นี้เก็บ “แม่แบบ” ที่ใช้ทำสินค้า TPT แบบทำซ้ำเร็ว

- `templates/tpt-listing-template.md` แม่แบบเขียนหน้าขาย
- `templates/worksheet-pack-template.md` โครงหน้าใบงานต่อ 1 product
- `templates/quality-checklist.md` เช็กลิสต์ก่อน export/upload
- `templates/product-pack/` โครงโฟลเดอร์ 1 product (copy แล้วเริ่มทำได้เลย)
- `templates/credits-page-template.md` แม่แบบ credits page
- `templates/thumbnail-copy-template.md` แม่แบบข้อความบน thumbnail
