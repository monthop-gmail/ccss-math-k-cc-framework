# Quality Checklist (Before Upload)

- มาตรฐานใน title/description ตรงกับเนื้อหา (เช่น `K.CC.B.5` จริง)
- Prompt ไม่กำกวม และเหมาะกับระดับ Kindergarten
- ช่วงตัวเลขถูกต้อง (0–10 / 0–20 / 1–10)
- มี answer key ทุกหน้า
- ตัวเลข/คำสั่ง/แบบฝึกหัด ไม่มีพิมพ์ผิด
- Layout: margin safe, font อ่านง่าย, ไม่แน่นเกิน
- Export PDF แล้วตรวจ: หน้าไม่ขาด, รูปไม่เพี้ยน, สีพิมพ์ได้
