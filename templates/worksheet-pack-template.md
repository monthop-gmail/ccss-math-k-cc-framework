# Worksheet Pack Template (1 Standard)

ใช้เป็นโครงสำหรับทำ 1 product ให้จบในชุดเดียว

## Page Set Formula

1. Cover
2. Teacher notes / instructions (1 หน้า)
3. Practice pages (5–10 หน้า)
4. Review page (1 หน้า)
5. Answer key (1–3 หน้า)
6. Credits

## Practice Page Patterns (เลือก 2–4 แบบ แล้ววน)

- Count & write number
- Count & circle (เลือกคำตอบ)
- Count & color (ระบายให้ครบจำนวน)
- Draw to match (วาดเพิ่มให้เท่ากับ)
- Cut & paste (ถ้าเป็น printable activity)

## Difficulty Ladder (แนะนำ)

- Level 1: ของเรียง + ช่วงตัวเลขน้อย
- Level 2: ของกระจาย + จำนวนมากขึ้น
- Level 3: หลายกลุ่ม / มี distractor
