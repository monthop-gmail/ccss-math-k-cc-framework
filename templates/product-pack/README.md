# Product Pack Skeleton

โครงโฟลเดอร์สำหรับ 1 product (copy แล้วเปลี่ยน standard/theme/range)

## Folders

- `src/` สคริปต์เนื้อหา + ข้อมูลหน้า (ก่อนทำกราฟิก)
- `listing/` ข้อความหน้าขาย (Title/Description/Keywords)
- `export/` ไฟล์ส่งออก (PDF) - ไม่ commit
- `thumbnails/` ภาพหน้าปก/พรีวิว - ไม่ commit

## How To Use

1) Copy โฟลเดอร์นี้ไปไว้ใน `examples/<your-pack>/`
2) แก้ `src/CONTENT.md`
3) แก้ `listing/listing.md`
